# Bryan's Photography Portfolio

A minimalist, high-end automotive and lifestyle photography portfolio website.

## Features
- **Cinematic Landing Page**: Features a "Lens Expansion" entry animation.
- **Dynamic Gallery**: Visual storytelling with hover effects and smooth transitions.
- **Custom Interaction**: Custom cursor and scroll-synchronized animations.
- **Performance Optimized**: Lazy loading, content-visibility, and smooth scrollbar integration.

## Tech Stack
- HTML5
- CSS3 (Vanilla)
- JavaScript (Vanilla)
- Google Fonts (Inter)

## Getting Started
1. Clone the repository.
2. Open `index.html` in any modern web browser.
3. To customize, replace images in the `assets/` folder and update `index.html`.

## Deployment
The project is ready to be hosted on any static site hosting service like GitHub Pages, Vercel, or Netlify.
